# AI Agents Demo Use-Cases

## Requirements
- Python 3.10 or higher
- Git


### Clone the Repository
```bash
git clone https://github.com/pt-kunal-mishra/AI_AGENTS_LLM.git
cd AI_AGENTS_LLM
```

### Installation
```bash
pip install -r requirements.txt
```
